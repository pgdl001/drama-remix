# Drama Remix Tool

Short drama remix batch production tool - generates thousands of unique derivative video works from master materials, with built-in platform review compliance.

## Quick Start

### Backend
```bash
cd backend
pip install -e .
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:5173 in your browser.
